# Kakko

カッコはCommon Lispのちょっとした方言です。
その構成要素は "(" と ")" です。
コンセプトなので技術的に面白いものでは無い。
sample.kakkoのすばらしい括弧をご覧下さい。

## Usage

```
(kakko:eval-kakko "((()()()()()()()()()()()()()()()()()()()()()()()()()()
                   ()()()()()()()()()()()()()()()()()()()()()()()))")
; => 1
```
